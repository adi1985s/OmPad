      OmPad

      What is OmPad?

      OmPad is a notepad to store notes, texts or even poems. It is focused in simplicity and speed for quick note taking while inspiration is at its peak.

      • Themes: Choose different themes in the top left menu or add your own ones easily through CSS
      • Pages: 5 pages to save different topics at the bottom left
      • Saves automatically: Just type, if you close the browser it will still be there

      About the app

      OmPad is build in HTML5/jQuery. It is a great example on how modern websites can be so tiny yet so functional. Great for students and curious developers. You can grab the code and create your own version and styles. All the notes and settings are saved through HTML5 LocalStorage, this allows you to use OmPad also while being offline.
